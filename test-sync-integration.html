🚀 快速开始

1. 启动服务端

# Windows
cd server
.\start.bat

# Linux/Mac
cd server
./deploy.sh local

2. 启动前端

# 在项目根目录
npm run dev

# 访问设置页面
# 点击左侧菜单 -> 设置 -> 同步设置

3. 配置服务端

在同步设置页面中：

  • 点击"配置"按钮
  • 设置服务器地址: http://localhost:3001
  • 设置WebSocket地址: ws://localhost:3001/ws
  • 点击"保存"

4. 注册/登录

在同步设置页面中：

  • 填写用户名、邮箱、密码
  • 点击"注册"或"登录"
  • 查看连接状态变为"已连接"
